# EMI Count - Online

## Overview

EMI Count - Online is a web-based EMI calculator designed to help users estimate loan repayments quickly and accurately.

The application calculates Equated Monthly Installments (EMI), total interest payable, and total repayment amount based on user-provided loan details.

It provides a simple and responsive interface for financial planning and loan comparison.

---

## Features

### EMI Calculation

Calculate monthly loan installments instantly.

### Interest Calculation

Determine total interest payable over the loan period.

### Total Repayment Estimation

Calculate the complete repayment amount including principal and interest.

### Responsive Design

Works across desktop, tablet, and mobile devices.

### Instant Results

Performs calculations in real time without page reloads.

### User-Friendly Interface

Simple input fields and clear result presentation.

## Formula Used

```text
EMI = P × R × (1 + R)^N / ((1 + R)^N − 1)
```

Where:

* P = Loan Amount
* R = Monthly Interest Rate
* N = Number of Monthly Installments

1. Clone or download the repository.

## Usage

1. Enter loan amount.
2. Enter annual interest rate.
3. Enter loan tenure.
4. Click Calculate.
5. View:

   * Monthly EMI
   * Total Interest
   * Total Repayment Amount
